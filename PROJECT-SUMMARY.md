## 技术特性

### 1. Vue 3 Composition API
- 使用 `<script setup>` 语法
- 响应式数据管理（ref、reactive）
- 生命周期钩子
- 计算属性和侦听器

### 2. Element Plus 组件库
- Table（表格）- 复杂数据展示
- Form（表单）- 数据录入和验证
- Dialog（对话框）- 弹窗交互
- Upload（上传）- 文件上传
- Pagination（分页）- 数据分页
- DatePicker（日期选择器）
- Rate（评分）- 星级评分
- Progress（进度条）- 销售进度
- Tag（标签）- 状态标识

### 3. ECharts 数据可视化
- 折线图（用户注册趋势）
- 柱状图（订单量趋势）
- 横向柱状图（热门剧目排行）
- 饼图（销售额分布）
- 柱状图（订单状态分布）

### 4. Pinia 状态管理
- 用户状态管理
- Token管理
- 登录状态管理
- 本地存储同步

### 5. Vue Router 路由管理
- 路由懒加载
- 路由守卫（权限控制）
- 动态路由参数
- 面包屑导航

### 6. 响应式设计
- Flex布局
- Grid布局
- 媒体查询
- 移动端适配

## 如何运行

### 1. 安装依赖
```bash
cd music-ticket-system-frontend
npm install
```

### 2. 开发模式
```bash
npm run dev
```
访问 http://localhost:5173

### 3. 构建生产版本
```bash
npm run build
```

### 4. 预览生产版本
```bash
npm run preview
```
